# Shop Billing System

Десктопная система выставления счетов для магазина.

## Технологии
- Python 3
- Tkinter
- MySQL

## Структура проекта
- `main.py` — точка входа
- `config.py` — конфигурация
- `database.py` — работа с БД
- `login.py` — авторизация
- `dashboard.py` — главное меню
- `products.py` — управление товарами
- `billing.py` — модуль продаж и чеков
